# FinSQL Implementation

**Status**: Not yet implemented

## Overview
FinSQL is a model-agnostic LLM-based text-to-SQL framework designed for financial analysis. It uses:
- Hybrid data augmentation (CoT, Synonyms, Skeleton)
- Parallel schema linking with cross-encoders
- Database value retrieval (MinHash-LSH + semantic matching)
- LoRA plugin hub with weight merging
- Output calibration (non-execution-based)
