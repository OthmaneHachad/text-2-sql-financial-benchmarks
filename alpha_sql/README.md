# Alpha-SQL Implementation

**Status**: Not yet implemented

## Overview
Alpha-SQL is a zero-shot text-to-SQL framework using Monte Carlo Tree Search (MCTS). It features:
- Progressive SQL construction via MCTS
- LLM-as-Action-Model
- Self-supervised rewards
- 7 action types with ordering constraints
